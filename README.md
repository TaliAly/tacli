# Tacli - Midu Hackton 2024

## Folder Structure:

- dist - development builds && outputs of typescript
- src/ - rest of the app
- src/sdk - vercel sdk AI handler

## How to run?

```bash
git clone https://github.com/TaliAly/tacli.git
# ssh version: git clone git@github.com:TaliAly/tacli.git
pnpm i
pnpm build

```
